# Carbon Kicker Landing Page
A repository to host landing page of Carbon Kicker.

## About Carbon Kicker
Carbon Kicker's mission is to help you reduce carbon emissions.

## How To Run?
- Clone repository using `git clone https://github.com/hifarhanali/carbon-kicker`
- Open terminal and go to the repository folder using `cd` command
- Run `yarn install`
- Run `yarn run start`

> This will open a new tab in the browser at [here](http://localhost:5173/)
